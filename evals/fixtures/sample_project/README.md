# Sample Project

A small Python project used as the fixture codebase for igni's
editor-agent eval suite. Key files:

- `src/auth/login.py`, `src/auth/token.py` — authentication
- `src/utils/format.py` — string helpers
- `src/parse.py` — config / int parsing
- `config/loader.py` — JSON config loader
- `tests/test_*.py` — pytest tests

This README contains a deliberate typo ("instalation") for the
`direct_one_line_typo_fix_no_spawn` eval case to fix.

## Instalation

Run `pip install -e .` to install in editable mode.
